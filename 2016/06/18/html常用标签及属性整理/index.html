html常用标签及属性整理

By 威威小威

发表于 2016-06-18

文章目录
  1. 1. 1. 常用标签
  2. 2. 2. 常用属性

html的标签和属性很多，但常用的标签并不多，由于并不建议将样式写进html中(样式推荐写在与html分离的css文件中)，因此与样式相关的属性也可以忽略，我们只需要记住常用的标签和属性，剩下的就交给css和js吧

1. 常用标签

1. <html> <head> <body>  html文档基本结构标签
2. <h1>至<h6> html标题,从h1-h6依次减小，有两点需要注意：
    - 确保将 HTML heading 标签只用于标题,不要仅仅是为了产生粗体或大号的文本而使用标题,
      应使用<strong>来表示强调，而更多对字体的设置应该使用css控制；
    - 浏览器会自动地在标题的前后添加空行，默认情况下，HTML会自动地在块级元素前后添加一个额外的空行，
      比如段落、标题元素前后。
3. <div> <span> html中最常用的标签，<div>定义文档中的节或区域（块级）,<span>定义文档中的行内的小块或区域。
   <div>与 CSS 一同使用，<div> 元素可用于对大的内容块设置样式属性;
   <span>与 CSS 一同使用时，<span> 元素可用于为部分文本设置样式属性。
4. <a> 超链接标签，<a>标签需与herf属性配合使用，示例<a herf="www.baidu.com">百度</a>
5. <img> 图像标签，<img>标签需与src属性配合使用，src可以指定图像的位置也可以是图像的网络地址，
    示例<img src="img/loading.gif">
6. <table> 表格标签
   <tr> 表格行标签
   <th> 表格表头标签
   <td> 表格单元格标签
   此外，<thaed>定义表格的页眉;<tbody>定义表格的主体;<tfoot>定义表格的页脚;
7. <ul> 无序列表标签；<ol>有序列表标签；<li>列表项标签；
8. <form> 表单标签，相关标签还有<input>定义输入域，<textarea>定义多行输入文本域，
    <select>与<option>定义下拉框，<button>定义一个按钮

2. 常用属性

html的属性配合标签使用，我把属性分为三种：

  1. 为了将html元素与css、js关联的属性，最主要的两个：class与id,前者规定元素的类名，后者规定元素的唯一 id。
  2. 基本与html标签配对无法分离的属性，如（1）a标签与herf属性（2）img标签与src属性 （3）input标签与type、name、value属性
  3. 其他属性，这些属性完成了css/js的功能

记住前两种就够了，至于第3种，个人观点，在你还没有成为资深前端开发时，能不用就不用吧，这里的大部分属性本来也就是html不推荐使用的，html属性完成与css、js的关联就够了，元素样式和动作的事，还是交给专业处理的css与js吧

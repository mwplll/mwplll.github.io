1. 算法思路

快速排序也是一种分治递归算法，对数组S排序的基本算法由以下四步组成：
（1）如果S中元素个数为0或1，则返回；
（2）取S中任一元素v，称之为枢纽元；
（3）将S中除v外的元素划分成两个不想交的集合：S1中的元素都小于等于v,S2中的元素都大于等于v
（4）组成新的数组{S1,v,S2},对S1,S2递归调用快速排序

图例：

上面只是展示了基本思路，为了更高效地实现快速排序算法，还需要特别注意枢纽元的选取和分割策略

1.1 枢纽元的选取

理论上无论选择哪个元素都能作为枢纽元都能完全排序工作，但是有些选择显然优于其他选择。

（1）选择第一个元素作为枢纽元，或者前两个互异的关键字中较大的作为枢纽元。
这是一种错误的做法，对于预排序的序列用这种方法将导致花费的时间为二次的。

（2）随机选取枢纽元
这是理想的枢纽元的选择办法，但是随机数的生成本身一般开销很大，所以不可取。

（3）三数中值分割法
取数组的最左边、最右边和中间的数，三个数的中值为枢纽元，这是常用的做法

1.2 分割策略

分割阶段要做的就是把所有小元素移到数组的左边，而把所有大元素移动数组的右边

介绍一种证明可行的分割方法，步骤为：
（1）将枢纽元和最后一个元素交换使得枢纽元离开要分割的数据段，i指向第一个元素，j指向倒数第二个元素。
（2）将i右移，移过小于枢纽元的元素，j左移，移过大于枢纽元的元素
（3）当i和j停止时，交换i和j指向的元素，继续第二步
（4）当i和j交错（i>j），不再交换，将枢纽元和i所指向的元素交换

一个重要的细节：当遇到与枢纽元相等的元素的处理，我们采用的办法是i和j都停止，仍然交换i和j指向的元素，为什么？都不停止的策略会导致出现两个不均衡的数组，如果所有关键字是相同的，则运行时间为O(N^2)

1.3 小数组的处理

对于小数组(N<=20)，快速排序不如插入排序。这里需要一个截止范围，一般取N=10，即数组长度小于10用插入排序，大于等于10则用快速排序

1.4 算法分析

快速排序的最坏运行时间为O(N^2),当待排序数组本身为有序或者为逆序；
最好运行时间为O(N log N)， 平均运行时间为O(N log N)。

2. java代码

源码：https://github.com/mwplll/algorithms

package sort;

public class QuickSort {
    private static final int CUTOFF = 10;

    /**
     * 快速排序的驱动程序
     * @param a
     */
    public static <AnyType extends Comparable<? super AnyType>>
    void quickSort(AnyType[] a){
        quickSort(a,0,a.length-1);
    }

    /**
     * 三数取中值，将最小者放到a[left],最大者放到a[right],中位数放到a[right-1],为分割阶段做准备
     * @param a
     * @param left
     * @param right
     * @return
     */
    public static <AnyType extends Comparable<? super AnyType>>
    AnyType median3(AnyType[] a,int left,int right){
        int center = (left+right)/2;
        if(a[center].compareTo(a[left])<0)
            swapReferences(a,left,center);
        if(a[right].compareTo(a[left])<0)
            swapReferences(a,left,right);
        if(a[right].compareTo(a[center])<0)
            swapReferences(a,center,right);

        //将三数的中值放到倒数第二个位置上
        swapReferences(a,center,right-1);
        return a[right-1];
    }

    /**
     * 将两个节点的值交换
     * @param a
     * @param i
     * @param j
     */
    private static <AnyType extends Comparable<? super AnyType>>
    void swapReferences(AnyType [] a,int i,int j){
        AnyType tmp;
        tmp = a[j];
        a[j] = a[i];
        a[i] = tmp;

    }

    /**
     * 快速排序主函数
     * @param a
     * @param left
     * @param right
     */
    private static <AnyType extends Comparable<? super AnyType>>
    void quickSort(AnyType[] a,int left,int right){
        if(left+CUTOFF<=right){
            //获取枢纽元
            AnyType pivot = median3(a,left,right);

            //开始分partitioning
            int i = left,j = right-1;
            for(;;){
                while(a[++i].compareTo(pivot)<0){}
                while(a[--j].compareTo(pivot)>0){}
                if(i<j)
                    swapReferences(a, i, j);
                else
                    break;
            }

            swapReferences(a, i, right-1); //交换枢纽元

            quickSort(a, left, i-1); //快排比枢纽元小的数组
            quickSort(a, i+1, right); //快排比枢纽元大的数组
        }
        else{//对于小数组，采用插入排序
            insertSort(a);
        }
    }

    /**
     * 插入排序
     * @param a
     */
    private static <AnyType extends Comparable<? super AnyType>>
    void insertSort(AnyType[] a){
        if(a!=null){
            for(int i=1;i<a.length;i++){
                AnyType temp = a[i];
                int j = i;
                //将a[i]插入到有序排列a[0..i-1]中
                while(j>=1&&a[j-1].compareTo(temp)>0){
                    a[j]=a[j-1];
                    j--;
                }
                a[j] = temp;
            }
        }

    }

    public static void main(String[] args) {
        Integer[] a = {38,65,97,76,13,27,49,23,1,25,18,100,0};
        quickSort(a);
        for(int i=0;i<a.length;i++){
            System.out.print(a[i]+" ");
        }
    }

}

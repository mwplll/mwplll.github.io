排序（五）-堆排序

1. 算法思路

构建N个元素的二叉堆，然后进行N次deleteMin操作，得到一个有序序列。

注意点：

  • 堆有大顶堆和小顶堆之分。大顶堆是指分支节点大于它的所有子节点，堆里最大元素是根节点；小顶堆是指分支节点小于它的所有子节点，堆里最小元素是根节点。

  • 大多数算法介绍都是以1为数组的起始下标的，左右叶子节点的计算分别为：lchild=2parent; rchild =2parent+1；而实际编程语言大多数都是以0为起始下标的，所以子节点计算需要调整为：lchild=2parent+1; rchild=2parent+2。

  • 算法总的运行时间为O(N log N)。

  • 该算法的主要问题在于它使用了一个附加的数组，因此，存储空间增加了一倍。回避使用第二个数组的办法：考虑在每次deleteMin之后，堆缩小1，因此，位于堆中最后的单元可以用来存放刚刚删去的元素。

  • 从堆顶取出最大元素与最后一个堆元素交互后，堆的大小减1。所以堆在排序过程中是逐渐缩小的，但根总是第一个元素，所以可以利用之前也构建好的堆，只需要调整一棵子树。

2. java代码

package sort;

public class HeapSort {
    private static int leftChild(int i){
        return 2*i+1;
    }

    /**
     * 下滤
     * @param a 数组
     * @param i 进行下滤的节点的位置，其中以根节点为0
     * @param n 二叉堆的长度
     */
    private static <AnyType extends Comparable<? super AnyType>>
    void percDown(AnyType [] a,int i,int n){
        int child;
        AnyType tmp;

        for(tmp=a[i];leftChild(i)<n;i=child){
            child = leftChild(i);
            if(child!=n-1 && a[child].compareTo(a[child+1])<0)
                child++;
            if(tmp.compareTo(a[child])<0)
                a[i] = a[child];
            else
                break;
        }
        a[i] = tmp;
    }

    /**
     * 将两个节点的值交换
     * @param a
     * @param i
     * @param j
     */
    private static <AnyType extends Comparable<? super AnyType>>
    void swapReferences(AnyType [] a,int i,int j){
        AnyType tmp;
        tmp = a[j];
        a[j] = a[i];
        a[i] = tmp;

    }

    public static <AnyType extends Comparable<? super AnyType>>
    void heapSort(AnyType[] a){
        //构建堆
        for(int i = a.length/2;i>=0;i--)
            percDown(a, i, a.length);
        for(int i = a.length-1;i>0;i--){
            swapReferences(a,0,i); //将删除的最大值放到最后一个节点，将原来的最后一个节点放到根节点
            percDown(a, 0, i);
        }
    }

    public static void main(String[] args) {
        Integer[] a = {38,65,97,76,13,27,49};
        heapSort(a);
        for(int i=0;i<a.length;i++){
            System.out.print(a[i]+" ");
        }
    }

}

排序（六）-归并排序

发表于 2015-08-28

1. 算法思路

这个算法的基本操作是合并两个已排序的表，因为这两个表是已排序的，所以若将输出放到第3个表中，则该算法可以通过对输入数据一趟排序完成。基本的合并算法是取两个数组A和B,一个输出数组C,已经三个计数器Actr、Bctr、Cctr,它们的初始置于对应数组的开始端。A[Actr]和B[Bctr]中的较小者被拷贝到C中下一个位置，相关的计数器推进一步。当两个表中有一个用完的事后，则将另一个表中剩余部分拷贝到C中。

归并算法:（1）如果N=1，那么只有一个元素需要排序；
（2）否则，递归地将前半部分数据和后半部分数据各自归并排序，得到排序后的两部分数据，然后再将这两部分合并到一起

该算法是经典的分治策略，它将问题分成一些小的问题然后递归求解，而治的阶段则将分的阶段解得的各答案修补在一起。

算法分析：

  • 进行合并操作的复杂度为O(n)，由于每次把数组分为2个子数组，需要进行lg(n)次合并，所以时间复杂度为O(n*lg(n))，。

  • 每次合并都需要与两个子数组长度和 大小的数组，所以空间复杂度为O(n)。

2. java代码

package sort;

public class MergeSort {
    /**
     * 类内部方法，合并两个有序表
     * @param a 待排序数组
     * @param tmpArray 用于交换的中间数组
     * @param leftPos 子表中左边部分起始位置
     * @param rightPos 字表中右边部分起始位置
     * @param rightEnd 字表中右边部分结束位置
     */
    private static <AnyType extends Comparable<? super AnyType>>
    void merge(AnyType [] a,AnyType [] tmpArray,int leftPos,int rightPos,int rightEnd){
        int leftEnd = rightPos-1;
        int tmpPos = leftPos;
        int numElements = rightEnd-leftPos+1;

        while(leftPos <= leftEnd && rightPos <= rightEnd)
            if(a[leftPos].compareTo(a[rightPos])<=0)
                tmpArray[tmpPos++] = a[leftPos++];
            else
                tmpArray[tmpPos++] = a[rightPos++];

        //拷贝左表中剩余部分
        while(leftPos<=leftEnd)
            tmpArray[tmpPos++] = a[leftPos++];

        //拷贝右表中剩余部分
        while(rightPos<=rightEnd)
            tmpArray[tmpPos++] = a[rightPos++];

        //将中间表的数据拷贝回待排序表
        for(int i=0;i<numElements;i++,rightEnd--)
            a[rightEnd] = tmpArray[rightEnd];        
    }

    /**
     * 将待排序表a中从a[left]到a[right]的元素进行排序，中间表依赖tmpArray
     * @param a
     * @param tmpArray
     * @param left
     * @param right
     */
    private static <AnyType extends Comparable<? super AnyType>>
    void mergeSort(AnyType [] a,AnyType [] tmpArray,int left,int right){
        if(left<right){
            int center = (left+right)/2;
            mergeSort(a,tmpArray,left,center);
            mergeSort(a,tmpArray,center+1,right);
            merge(a,tmpArray,left,center+1,right);
        }

    }

    public static <AnyType extends Comparable<? super AnyType>>
    void mergeSort(AnyType [] a){
        AnyType [] tmpArray = (AnyType[]) new Comparable[a.length];
         mergeSort(a,tmpArray,0,a.length-1);
    }


    public static void main(String[] args) {
        Integer[] a = {38,65,97,76,13,27,49};
        mergeSort(a);
        for(int i=0;i<a.length;i++){
            System.out.print(a[i]+" ");
        }
    }
}

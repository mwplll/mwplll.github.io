指针

  1. *p的区别
    int *p;
*p = 10;
    这里的两个p，尽管形式相同，但含义是完全不同的。前者定义了指针变量，p是变量名，表示其后的变量是指针；后者*p代表指针p所指向的变量。
  2. 指针只有被赋值后才能正确被使用。指针如果没有被赋值，他的值是不确定的，即它指向一个不确定的单元，使用这样的指针，可能会出现难以预料的错误。为了避免引用未赋值的指针造成危害，在定义指针时，可先将它的初始值置为空，如*p=NULL;
  3. 指针每一次加1或者减1，并非指针的值加1或者减1，而是加上或者减去该指针所指向的那个变量数据类型的长度，即它指向的存储单元所占用的字节数。
  4. 在函数定义时，被声明为数组的形参实际上是一个指针。当传递数组时，按值调用传递它的基地址，数组元素本身不被复制。

  5. scanf()和gets()函数的区别：scanf()只能输入不能带有空格的字符串，gets()可输入空格，遇回车符输入结束；

    printf()和puts()的区别：后者输出字符串会自动换行。

  6. calloc与malloc动态内存分配函数。其中calloc在动态分配内存后，自动初始化该内存空间为0，而malloc不会。

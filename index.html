面试经历--蘑菇街

发表于 2015-05-06   |   分类于 求职笔试面试   |  

今天上午9点去参加了蘑菇街的面试，去面试前，已向同学了解到，蘑菇街校招实习生面试一般会三面一次性进行，两次技术面，一次HR面，如果某一面没过，就会直接让你回来了。我在技术一面后就让回来等通知，应该是被刷了，就当是一次经验积累吧。
刚到蘑菇街公司，在前台会让你填一张个人简要信息的表格，过会面试官过来把这张表格和你在网上提交的简历一起拿去，去11楼一个小房间开始面试。
面试的内容主要以项目为主，从项目展开，问相关的知识点，问得很深，他会抓住一个点，一直不断地问，直到你答不出来为止。
我做过一个家装O2O的网站，自我介绍后，就是让我介绍一下这个项目。介绍完了根据项目问的主要问题是：

  1. 采用了DapperPHP框架，请说明采用这个框架的原因。
  2. 给这个框架增加了Lib层，请说明原因
  3. 团队人员组成，每个人分别是做什么的（这个问题面试官主要还是想了解你在这个项目中做了什么，发挥了什么作用）
  4. 当我说后台的数据库设计是我设计的，他会问为什么要那样设计，要求当场画ER图；还问订单部分的数据库设计，同样要求是当场画ER图
  5. 中间提到数据库优化，要求介绍一下具体做了哪些优化，为什么要做那些优化，优化后实际的效果是什么，还会要求当场写SQL语句。
  6. 问我在数据库方面看了那些书。（平时书看得少就惨了，我说只看了mysql必知必会，网络上搜索的知识点比较多）

  7. 问在程序开发中最不喜欢的事是什么？（这个问题当时答得太nc了，被问了一个多小时，这个时候已经完全不知道自己在说什么了，居然说不喜欢程序调试时很烦，有时逻辑是对的，但上板调试又出错。答完了之后直接就例行公事问我有什么想问他的，就让我回去等通知了）

    总体来说，面试是以项目展开，知识点会问得很细，很深，所以准备这种面试必须将自己做的项目各个方面研究透，不能被问倒。另外，像问程序开发中最不喜欢的事，千万不能老老实实回答说不喜欢繁琐了，或许可以说沟通不充分，信息不透明，比如我们实验室（实验室的项目是秘密级的）

数据结构第1章--绪论

发表于 2015-05-05   |   分类于 数据结构   |  

1.1 什么是数据结构

数据结构描述现实世界实体的数学模型（非数值计算）及其上的操作在计算机中的表示和实现。

1.2 基本概念和术语

  1. 数据：所有能被输入到计算中，且被计算机处理的符号的集合。计算机操作的对象的总称。
  2. 数据元素：数据的基本单位。
  3. 数据项：数据结构中讨论的最小单位，数据元素是数据项的集合。
  4. 数据结构：相互之间存在一种或多种特定关系的数据元素的集合。
    关系：1）线性结构 2）树形结构 3）图状结构 4）集合结构
    数据结构的形式化定义：Data_Structure = (D,S)
    D是数据元素的有限集，S是D上关系的有限集。
  5. 数据结构在计算机中的表示（映像）：
    1）数据元素的映像表示：二进制位串
    2）数据关系的映像表示：顺序映像（以存储位置的相邻表示后继关系），链式映像（以附加信息—指针表示后继关系）
  6. 数据类型
    数据类型是一个值的集合和定义在这个值集上的一组操作的总称
  7. 抽象数据类型（ADT）
    指一个数学模型以及定义在该模型上的一组操作
    两个重要特征：
    1）数据抽象
    2）数据封装
    描述方法：（D,S,P）P为对D的操作集

1.3 算法和算法分析

  1. 算法
    算法是为了解决某类问题而规定的一个有限长的操作序列。
    5个重要特征：1）有穷性 2）确定性 3）可行性 4）输入 5）输出
  2. 算法设计的原则
    1）正确性 2）可读性 3）健壮性：处理出错的方法不应该是中断程序的进行，而应是返回一个表示错误或错误性质的值，以便在更高的抽象层次上进行处理 4）效率与低存储量需求
  3. 算法效率的衡量方法和准则
    时间复杂度 T(n)=O(f(n))
    空间复杂度 S(n)=O(f(n))
  4. 如何估算时间复杂度
    算法 = 数据结构 + 原操作（固有数据类型的操作）
    算法的执行时间 与 原操作的执行次数 成正比，因此可以选取一种基本操作的原操作，以基本操作在算法中的重复执行的次数，作为算法运行时间的衡量准则。
    如选择排序，时间复杂度O(n^2)
    冒泡排序，最好的情况下为0，在最坏情况下时间复杂度O(n^2)

leetcode(1)-Two Sum

发表于 2015-04-27   |   分类于 leetcode   |  

题目

Given an array of integers, find two numbers such that they add up to a specific target number.

The function twoSum should return indices of the two numbers such that they add up to the target, where index1 must be less than index2. Please note that your returned answers (both index1 and index2) are not zero-based.

You may assume that each input would have exactly one solution.

Input: numbers={2, 7, 11, 15}, target=9
Output: index1=1, index2=2

分析

1、暴力求解，直接做两层循环找出两个数所在的位置，但时间复杂度为O(n^2)，会超时
2、先做一个快排

/*
 1、暴力求解，时间复杂度为O(n^2),会超时: Time Limit Exceeded
 */
int* twoSum1(int* nums, int numsSize, int target) {
    int *a = (int *)malloc(2*sizeof(int));
    int *index=(int *)malloc(numsSize*sizeof(int));
    int i=0;
    int j=0;
    for(i=0;i<numsSize;i++){
        index[i] = i+1;
    }
    for(i=0;i<numsSize;i++){
        for(j=i+1;j<numsSize;j++){
            if(nums[i]+nums[j] == target){
                a[0] = index[i];
                a[1] = index[j];
                break;
            }
        }
    }
    return a;
}

指针

  1. *p的区别
    int *p;
*p = 10;
    这里的两个p，尽管形式相同，但含义是完全不同的。前者定义了指针变量，p是变量名，表示其后的变量是指针；后者*p代表指针p所指向的变量。
  2. 指针只有被赋值后才能正确被使用。指针如果没有被赋值，他的值是不确定的，即它指向一个不确定的单元，使用这样的指针，可能会出现难以预料的错误。为了避免引用未赋值的指针造成危害，在定义指针时，可先将它的初始值置为空，如*p=NULL;
  3. 指针每一次加1或者减1，并非指针的值加1或者减1，而是加上或者减去该指针所指向的那个变量数据类型的长度，即它指向的存储单元所占用的字节数。
  4. 在函数定义时，被声明为数组的形参实际上是一个指针。当传递数组时，按值调用传递它的基地址，数组元素本身不被复制。

  5. scanf()和gets()函数的区别：scanf()只能输入不能带有空格的字符串，gets()可输入空格，遇回车符输入结束；

    printf()和puts()的区别：后者输出字符串会自动换行。

  6. calloc与malloc动态内存分配函数。其中calloc在动态分配内存后，自动初始化该内存空间为0，而malloc不会。

数据类型

  1. 数据类型转化
    字节小的可以向字节大的自动转换，但字节大的不能向字节小的自动转换
    如：字符类型可自动转化为整型，整型可自动转化为浮点类型
  2. char类型转化为int类型遵循ASCII码的对应值
    ASCII码表
  3. 强制类型转换
    强制转化后的运算结果不遵循四舍五入原则;
    转换后不会改变原数据的类型及变量值，只在本次运算中临时性转换。

运算符

  1. 如果相除的两个数都是整数的话，则结果也为整数，小数部分省略;而两个数中有一个为小数，结果则为小数，如：9.0/2 = 4.500000
  2. 取余运算中注意：该运算只适合用两个整数进行取余运算，如：10%3 = 1；而10.0%3则是错误的；运算后的符号取决于被模数的符号，如(-10)%3 = -1;而10%(-3) = 1。
  3. 运算符优先级，（）最高
  4. 闰年计算方法，（1）普通年份能被4整除但不能被100整除；（2）后两位为0的年份能被400整除；则为闰年 y%400==0||(y%4==0&&y%100!=0)

结构语句

  1. c语言中for中使用的变量一定要在循环之前定义。如以下的用法就是错误的：

    //各表达式中的变量一定要在for循环之前定义
for(int num=0;num<=10;num++){
    printf('%d\n',num);
}

    相对的，在C++中，以上表达式就是正确的。

  2. 在实际开发中一般最多用到三层重循环；

  3. 在多层循环中，一个break语句只能跳出当前循环；
  4. switch语句，后面表达式语句只能是整型或者字符类型；在case后，允许有多个语句，可以不用{}括起来。

函数

  1. void函数中可以有执行代码块，但是不能有返回值，另void函数中如果有return语句，该语句只能起到结束函数运行的功能。其格式为：return;

  2. 在复合语句中也可以定义局部变量，其作用域只在复合语句内部。

    #include <stdio.h>
int x = 77;
void fn1()
{
    printf("fn1(): x=%d\n", x);       
}
int main()
{
    int x = 10;
    if(x>0)
    {
        int x = 100;
        x /= 2;
        printf("if语句内, x=%d\n", x);       
    }
    printf("main方法内, x=%d\n", x);
    fn1();
    return 0;
}

    输出结果为：

    if语句内, x=50
main方法内, x=10
fn1(): x=77

  3. C语言中存储类别分为四类：自动（auto）、静态（static）、寄存器的（register）和外部的（extern）。

  4. 静态局部变量属于静态存储类别，在静态存储区内分配存储单元，在程序整个运行期间都不释放；静态局部变量在编译时赋初值，即只赋初值一次；如果在定义局部变量时不赋初值的话，则对静态局部变量来说，编译时自动赋初值0（对数值型变量）或空字符（对字符变量）。

  5. 内部函数与外部函数。内部函数定义：

    static [数据类型] 函数名([参数])

    内部函数只能在其所处的源文件中使用，因此在不同的文件中出现相同的函数名称的内部函数是没有问题的。

数组

  1. 在声明数组后没有进行初始化的时候，静态（static）和外部（extern）类型的数组元素初始化元素为0，自动（auto）类型的数组的元素初始化值不确定。
  2. C语言不提供计算数组长度的方法；
  3. 数组名作为函数实参传递时，函数定义处作为接收参数的数组类型形参既可以指定长度也可以不指定长度。
  4. C语言中是不能直接定义字符串数据类型的。字符串定义方法：
    （1）、char 字符串名称[长度] = "字符串值";
（2）、char 字符串名称[长度] = {'字符1','字符2',...,'字符n','\0'};
    采用第2种方式的时候最后一个元素必须是’\0’，’\0’表示字符串的结束标志；
    采用第2种方式的时候在数组中不能写中文。
  5. 常用的字符串函数

  6. 多维数组初始化：
    1、数据类型 数组名称[][]...[常量表达式n] = <!--￼0-->;
